Hi, how can we help?



Tutor applications?

Purpose:To see a list of all the tutor applications on the system.

How to: Navigate to the tutor application option on the adminhome side navigation bar. The system will display a table with all the tutor applications with their relevant details and downloadable cv.

Accept/reject Tutor application?

Purpose: To accept or reject a tutor application.

How to: Navigate to a specific tutor application by means of the tutor application icon on the side nav bar, then either click the accept or reject button. Both options will require confirmation by clicking the yes button on the prompt.

Create Tutor?

Purpose: To give a tutor access to the system.

How to: When selecting the accept option on a tutor application and confirming the acceptance of the applicant, the system will redirect you to a create tutor pop-up where they enter a temporary username and password that is sent via email to the tutor for them to gain access to the system and set up their profile.


Delete Tutor?

Purpose: To delete tutor from the system.

How to: Navigate to the tutors screen from the side navigation bar on the admin home screen by means of clicking the tutors icon. Select the delete button on the row of the tutor that you want to delete. A confirmation message will appear where you can confirm the deletion with the click of the yes button.

Create University?

Purpose: To add a university to the system.

How to: Navigates to the University screen by selecting the Universities tab in the side nav bar. Select the create university button and a pop-up appears, enter the new university name into the input field and selects the submit button. The system generates a confirmation pop-up modal and confirm by selecting yes.

Update University?

Purpose: To edit a university on the system.

How to: Navigate to a specific university by means of clicking the university icon on the side nav bar and clicks the edit button, then edits the input field provided and clicks the submit button. The system prompts for confirmation and confirm by selecting yes.


Delete University?

Purpose:To delete a university on the system.

How to: Navigate to a specific university by means of clicking the university icon on the side nav bar and clicks the delete button. The system prompts for confirmation and give it by clicking the yes button on the pop-up.
